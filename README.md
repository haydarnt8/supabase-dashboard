# Supabase Dashboard

لوحة تحكم Vue.js 3 مع Pinia و Vuetify لإدارة الدورات والمواقع والامتحانات.

## المميزات

- ✅ Vue 3 مع TypeScript
- ✅ Pinia لإدارة الحالة
- ✅ Vuetify 3 مع دعم RTL كامل
- ✅ Supabase للخلفية
- ✅ المصادقة بالبريد الإلكتروني وكلمة المرور
- ✅ عمليات CRUD كاملة للدورات، المواقع، والامتحانات
- ✅ إدارة المستخدمين
- ✅ الوضع الداكن
- ✅ تصميم متجاوب

## البنية

```
src/
├── components/       # مكونات Vue القابلة لإعادة الاستخدام
├── views/           # صفحات التطبيق
├── stores/          # مخازن Pinia
├── router/          # إعدادات Vue Router
├── plugins/         # إعدادات Vuetify
├── lib/            # عميل Supabase
└── types/          # أنواع TypeScript
```

## التثبيت

### المتطلبات

- Node.js 18+
- pnpm
- حساب Supabase

### الخطوات

1. **استنساخ المشروع**
   ```bash
   cd supabase-dashboard
   ```

2. **تثبيت الحزم**
   ```bash
   pnpm install
   ```

3. **إعداد متغيرات البيئة**

   انسخ ملف `.env.example` إلى `.env`:
   ```bash
   cp .env.example .env
   ```

   قم بتحديث ملف `.env` ببيانات مشروع Supabase الخاص بك:
   ```env
   VITE_SUPABASE_URL=https://your-project.supabase.co
   VITE_SUPABASE_ANON_KEY=your-anon-key
   ```

4. **تطبيق قاعدة البيانات**

   قم بتشغيل SQL Schema المتوفر في مشروع Supabase الخاص بك لإنشاء الجداول والسياسات.

5. **تشغيل التطبيق**
   ```bash
   pnpm dev
   ```

   سيعمل التطبيق على: `http://localhost:5173`

## البناء للإنتاج

```bash
pnpm build
```

## الاستخدام

### تسجيل الدخول

- استخدم بريد إلكتروني وكلمة مرور صالحة من Supabase Auth
- جميع المستخدمين المصادق عليهم لديهم صلاحيات المسؤول

### التنقل

1. **الصفحة الرئيسية (الدورات)**
   - عرض جميع الدورات
   - إضافة/تعديل/حذف الدورات
   - انقر على دورة للانتقال إلى مواقعها

2. **المواقع**
   - عرض المواقع لدورة محددة
   - إضافة/تعديل/حذف المواقع
   - انقر على موقع للانتقال إلى امتحاناته

3. **الامتحانات**
   - عرض الامتحانات لموقع محدد
   - إضافة/تعديل/حذف الامتحانات
   - جميع تفاصيل الامتحان (التاريخ، الوقت، الفصل، إلخ)

4. **إدارة المستخدمين**
   - عرض جميع المستخدمين
   - تعديل معلومات المستخدمين

### الميزات الإضافية

- **الوضع الداكن**: انقر على أيقونة الشمس/القمر في شريط التنقل
- **التنقل بالخبز**: يظهر في شريط التنقل العلوي لتتبع موقعك
- **البحث**: متوفر في صفحة إدارة المستخدمين

## التقنيات المستخدمة

- **Vue 3**: إطار عمل JavaScript التقدمي
- **TypeScript**: JavaScript مع الأنواع
- **Pinia**: إدارة الحالة لـ Vue
- **Vuetify 3**: مكتبة مكونات Material Design
- **Vue Router**: التوجيه الرسمي لـ Vue
- **Supabase**: منصة خلفية مفتوحة المصدر
- **Vite**: أداة بناء سريعة

## البنية الأمنية

- يتم التحكم في الوصول عبر Row Level Security (RLS) في Supabase
- المصادقة مطلوبة لجميع المسارات (ما عدا تسجيل الدخول)
- جميع المستخدمين المصادق عليهم لديهم صلاحيات الكتابة
